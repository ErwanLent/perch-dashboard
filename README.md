# Perch Dashboard
Contributor: Erwan Lent
<br />
[Live Example](http://dashboard.perchcard.com/)

#### Technologies Used:

* Vuejs front-end framework
* Mapbox-GL (vector mapbox)
* Highcharts
* Nodejs + Express
* Semantic UI

#### Installation
Recommended nodejs version > 8.1.0
```
npm install
node app.js
```
